# Hubot Bellako

**Hubot script que traduce de chileno a bèlláákó lang**

### Instalación:

````
$ npm install hubot-bellako --save
````

### Uso:

````
hubot en bellako <frase>
````

**Ejemplo:**

````
hubot> hubot en bellako hola como vas

hubot> hóláá cómó váás máZnáátèdííghó óèzíí
````
